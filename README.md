# Firebase Rest API
> Backend API for FireAuth application, which is a authentication app.

## Backend: 
Node js, Express js, Firebase, Nodemon, Dotenv, Cors, Colors js, morgan.

## Install Dependencies
```
npm install
```

## Run App
```
# Run in dev mode
npm start

# Run in production mode
npm start
```
